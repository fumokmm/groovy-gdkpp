How to use
----------

## 1. Compile 'groovy-extensions.jar'. (Require [Gradle][]) ##

> $ gradle

## 2. Put 'groovy-extensions.jar' on the classpath. ##

## 3. Using example ##

### ListExt ###
> ext.ListExt // Using ListExt.
>
> assert [[1, 4], [2, 5], [3, 6]] == [1,2,3].zip([4,5,6])

[gradle]: http://gradle.org/ "Gradle"
